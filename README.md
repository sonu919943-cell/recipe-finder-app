# 🍳 Recipe Finder – Search, Cook & Save

A simple and user-friendly web app that helps users find recipes quickly.
Built using **HTML, CSS, and JavaScript**, and powered by the **TheMealDB API**, this project focuses on clean design and smooth user experience.

---

## 🌐 Live Demo

🔗 https://the-recipes-finder.netlify.app

---

## 📌 About the Project

We often face a common problem —
👉 *“Aaj kya banaye?”*

Most recipe websites are either too cluttered with ads or require login even for basic features.
So, we built **Recipe Finder** — a lightweight and clean application where users can search recipes instantly without any distractions.

---

## 🚀 Features

* 🔍 Search recipes by name
* 🖼️ View recipes with images and details
* 📖 Step-by-step cooking instructions
* 🎬 YouTube video support (if available)
* ❤️ Save favorite recipes using localStorage
* 📱 Fully responsive design (mobile + desktop)

## ⚙️ How It Works

1. User enters a recipe name
2. Data is fetched from TheMealDB API
3. Recipes are shown as cards
4. Clicking a card opens full details
5. User can save recipes to favorites
6. Favorites are stored in browser storage

---

## 🧠 Why We Built This

While exploring recipe platforms, we noticed:
